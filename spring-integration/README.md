# Spring Integration with Annotations

This example, will explain how to use

 - @MessageGateway
 - @Gateway
 - @MessageEndPoint
 - @Router
    - PayLoadType Router
    - RecipientList Router
    - HeaderValue Router
 - @Filter
 - @Transformer
 - @ServiceActivator
 - @IntegrationComponentScan
 - @EnableIntegration
